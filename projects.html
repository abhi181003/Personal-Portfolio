Projects

Database Assignment 1

The Library Book Management System Database is a comprehensive solution designed to efficiently manage all aspects of a library's operations. It includes four key tables for books, issues, returns, and student records, with a backend powered by SQL Server Management Studio (SSMS) and a frontend in Microsoft Access. This database enables seamless cataloging, lending, and tracking of library materials while providing user-friendly interfaces for data input and retrieval. The guide outlines how to navigate the frontend application, emphasizing the importance of forms and reports for data manipulation and presentation, making it a valuable resource for both beginners and intermediate users.


Date Completed

18 October 2023


Technologies Used

Ms Access, SQL Server Management System

Web Programming Assignment 2

The Personal Website Development project involves creating a professional personal website using HTML, CSS, and JavaScript. It consists of four main pages: a landing page introducing the author, a resume page with contact information, work experience, skills, and education details, a projects page showcasing notable projects with a simple slideshow, and a contact page with a submission form. The project must adhere to good web design practices, accessibility guidelines, and coding standards, offering a clean, impressive, and responsive website to present the author's skills and ambitions effectively.

4 November 2023


Technologies Used

JavaScript, CSS, HTML
